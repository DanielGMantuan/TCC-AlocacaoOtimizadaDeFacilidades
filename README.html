COMPFLOR: Complemento para Otimização do Planejamento Florestal

Descrição

O COMPFLOR é um software desenvolvido para otimizar o planejamento do Manejo Florestal Sustentável (MFS). Ele utiliza técnicas de otimização como Simulated Annealing (SA) e o algoritmo Dijkstra para minimizar os custos operacionais e o impacto ambiental na exploração florestal. O objetivo é auxiliar gestores na tomada de decisões estratégicas, contribuindo para a redução de custos e a preservação das áreas florestais.

Funcionalidades Principais

  • Alocação otimizada de pátios de estocagem: Utiliza o método Simulated Annealing para determinar a localização ideal dos pátios, reduzindo distâncias e custos.
  • Definição de estradas florestais: Aplica o algoritmo Dijkstra para traçar as rotas mais curtas, conectando os pátios e minimizando a abertura de novas vias.
  • Planejamento de trilhas de arraste: Seleciona pontos de facilidade e conecta-os aos pátios, garantindo menor impacto ambiental e eficiência logística.
  • Geração de shapefiles: Cria arquivos geoespaciais para visualização clara da infraestrutura planejada.

Como Usar

  1. Execute o software: 
    python compflor.py
  2. Insira os dados de entrada (vertices, pontos de pátios, APP, etc.).
  3. Aguarde o processamento e visualize os resultados gerados.

Tecnologias Utilizadas

  • Python para a implementação dos algoritmos.
  • Sistemas de Informações Geográficas (SIG) para processamento de dados geoespaciais.
  • Simulated Annealing (SA) e Dijkstra para otimização.
